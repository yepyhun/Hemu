# Old Hermes Comparator

## Verdict

Old Hermes was not needed as an active oracle to localize the dominant seam in Phase 21.

## Why

- Core2 alone already falsified the strongest upstream hypotheses:
  - source absence
  - gross ingest/persist loss
  - session-local unsearchability
- The fixed-ten evidence now points mainly downstream to surface / handoff behavior, not to a missing legacy primitive.

## Phase 21 Rule

- old Hermes remains diagnostic-only
- no code was borrowed
- no architecture rollback was performed

## When Legacy Oracle Would Be Worth Reopening

- if a future milestone needs a direct comparison of:
  - prompt packaging
  - authoritative answer formatting
  - memory-tool handoff shape

That is a different question from memory presence/admission, and it should be handled in its own bounded milestone.
