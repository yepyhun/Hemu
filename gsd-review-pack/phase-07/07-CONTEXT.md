## Phase Boundary

Phase 7 is not another default-selection phase and not another tuning loop.

It may:
- reduce the residual `mixed_hold` result to one or two root blocked mechanisms
- identify where hybrid gains fail to reach the final answer path
- define one bounded breakthrough hypothesis
- define one local proof path and one stop rule

It may not:
- re-run paid subsets inside the phase
- tune per-question benchmark cases
- reopen substrate migration work
- reinterpret the Phase 6 measurement
- grow into broad implementation without a sharply bounded hypothesis

## User Steering

The user has already chosen the hybrid branch as the forward path. The remaining question is not "which branch wins" but "where is the biggest realistic leverage for a breakthrough, without falling back into endless micro-fixes?".

- which blocked mechanism most likely prevented the hybrid from turning `+1/70` into a more material gain
- which single bounded hypothesis is worth building next
- and what evidence would mean "stop, this is not breakthrough territory"
