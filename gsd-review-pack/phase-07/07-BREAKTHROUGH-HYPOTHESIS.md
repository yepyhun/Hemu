# Phase 7 Breakthrough Hypothesis

## Chosen Hypothesis

### Authoritative Eligibility Bridge For Covered Families

When the hybrid substrate returns stronger evidence for **already-covered** families, the routing layer should widen authoritative eligibility into the existing deterministic answer path more often, without introducing:
- new families
- softer semantic comparators
- benchmark-specific phrasing
- truth/state contract changes

## Why This Hypothesis Wins

This is the best leverage-to-complexity bet because:

1. it targets the dominant blocked mechanism from Phase 7
2. it is hybrid-specific rather than branch-independent cleanup
3. it can create multiplicative gain if more improved retrieval packets reach the final answer path
4. it stays inside an existing seam:
   - routing / eligibility
   - not substrate rewrite
   - not answer-surface style work
   - not comparator softening

## Why Other Candidates Were Rejected

### Comparator coverage expansion

Rejected as the primary breakthrough hypothesis because:
- it explains only `4` stable cases
- it is branch-independent
- it looks more like bounded cleanup than breakthrough

### More hybrid substrate work

Rejected for now because:
- Phase 6 already showed only a tiny end-to-end gain
- another substrate round before fixing answer-path propagation is likely to repeat the same plateau

### Case-by-case prompt tuning

Rejected because:
- it directly violates the anti-loop rules
- it would optimize the benchmark surface rather than the underlying leverage point

## Cheap Falsifiable Local Proof

Use a small frozen slice drawn from the unchanged `prompt_miss / judge: unknown` cluster.

The proof should measure:
- whether the proposed bridge increases authoritative-route eligibility
- whether those cases now enter an existing deterministic/local-comparable path
- whether this happens without:
  - new family growth
  - comparator softening
  - prompt phrasing hacks

## Success Shape

The hypothesis is worth building next only if the local proof shows a **material** shift in route eligibility on the representative slice.

If the bridge does not move those cases into the authoritative path, then the true bottleneck lies elsewhere and this hypothesis should be abandoned quickly.
