# Plan 20-03: Replay The Same Ten Cases And Decide

## Goal

Run local proof and a paid replay on the same fixed `10` cases, then produce one canonical forward-or-stop verdict.

## Tasks

- complete the forensic ledger for the frozen `10`
- show whether the chosen primitive borrow changes the suspected death seam
- run targeted local proof where appropriate
- run one paid replay on the exact same frozen `10`
- compare:
  - pass count
  - evidence-bearing movement
  - selector engagement
  - route-note movement
  - failure-family movement
- write one verdict only:
  - borrow moved the fixed `10` enough to justify continuation
  - or stop and do not widen this line further

## Deliverables

- `20-FORENSIC-LEDGER.jsonl`
- `20-TEN-CASE-GATE-STATUS.json`
- `20-TEN-CASE-ATTRIBUTION-STATUS.json`
- `20-OUTCOME.json`

## Verification

- the replay uses the exact frozen `10`, not a drifting subset
- the verdict is based on the paid replay, not only local proof
- if the replay still does not move enough, the outcome is a stop verdict
- no broader continuation is recommended without explicit replay movement
