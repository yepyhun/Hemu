# Phase 08.1 Boundary

## In Scope

- small Core2-native invariant catalog
- narrow acceptance harness for those invariants
- explicit structured-memory noise repair for obvious tool/file error artifacts

## Out Of Scope

- retrieval ranking borrow
- activation maintenance borrow
- native-memory mirroring expansion
- benchmark reruns
- comparator softening
- family proliferation

## Imported Seam

The legacy import is conceptual, not wholesale:

- architect rules become Core2 invariant specs and tests
- legacy repair semantics become a Core2-native reject/repair pass

No second truth engine or legacy runtime transplant is introduced.
