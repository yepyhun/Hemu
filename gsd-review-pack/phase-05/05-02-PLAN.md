# Plan 05-02: Define Canonical Outcome Artifacts

## Goal

Define the canonical machine-readable artifacts that baseline and hybrid runs must emit so later comparison and verdicting are deterministic.

## Tasks

1. Define the baseline outcome artifact shape.
2. Define the hybrid outcome artifact shape.
3. Define a canonical diff/outcome summary schema that later phases can consume directly.

## Verification

- Baseline and hybrid outcomes share the same comparable schema.
- The diff artifact can represent improvement, regression, or mixed results without prose-only interpretation.
- Later milestone phases can consume these artifacts without re-deriving raw meaning from benchmark logs.
