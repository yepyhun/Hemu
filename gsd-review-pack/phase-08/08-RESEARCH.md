# Phase 8: Authoritative Eligibility Bridge - Research

**Date:** 2026-04-07
**Scope:** planning guidance only

## What The Prior Milestone Already Proved

- The hybrid branch is directionally better than the archived baseline, but only by a narrow margin on the frozen broader comparison.
- The dominant residual bottleneck is not broad substrate weakness; it is that hybrid gains do not reach the authoritative answer path often enough.
- The highest-leverage next move is therefore a routing/eligibility intervention for **already-covered** families.

## Planning Implications

### What This Phase Should Build

Build one bounded bridge that widens authoritative eligibility when:
- the family is already covered
- the evidence packet is stronger under the hybrid path
- the deterministic answer path already exists

### What This Phase Must Not Build

- new families
- branch-independent cleanup disguised as breakthrough work
- broader comparator logic
- prompt/handoff wording hacks
- another benchmark protocol change

## Cheap, Honest Proof

The phase should prove value locally before any new broader paid run is discussed.

Recommended proof shape:
- freeze a tiny representative blocked-path manifest
- record whether each case entered the authoritative path before and after the bridge
- emit one canonical local outcome artifact with:
  - cases tested
  - authoritative-route delta
  - regressions, if any
  - verdict: `go`, `hold`, or `stop`

## Risks

### Biggest Failure Mode

The phase drifts into case-wise rescue work:
- widening eligibility differently per topic
- silently adding family-specific clauses
- patching benchmark-facing text rather than changing route eligibility

### Secondary Failure Mode

The phase “looks better” in traces but does not materially change authoritative-route entry on the frozen slice.

## Guardrails

- One bounded bridge only.
- One frozen blocked-path manifest.
- One canonical local outcome file.
- If route-shift is not real on the frozen slice, stop and do not schedule a broad rerun from this phase.
