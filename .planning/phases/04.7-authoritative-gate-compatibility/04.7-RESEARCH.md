- The correct move is therefore a small compatibility layer, not another structural memory expansion phase.

## Recommended Implementation Shape

Keep this phase small and mechanical:

1. Add a deterministic local comparator/canonicalizer for promptless authoritative answers.
2. Harden the gate runner so promptless short-circuit cases are recorded and classified explicitly.
3. Prove the result with a canary rerun and hand Phase 04.1 back a cleaner external gate path.

This preserves the current Core2 kernel boundary while making the external evaluation path less lossy.

## Testing Guidance

- Reuse the existing benchmark/authoritative tests rather than creating a second test stack.
- Add only the smallest new test coverage needed for:
  - canonical comparator behavior
  - promptless short-circuit artifact labeling
  - fail-closed `judge: unknown` classification
- The comparator must be tested against false-positive cases so “approximately right” wording cannot silently pass if the structured truth is different.
- If rendered text drops essential semantic context, treat that as a real handoff quality problem rather than assuming the external judge is simply wrong.
- Require local proof and canary proof before the next paid rerun.
- Keep the current 04.6 rerun discipline:
  - local handmade set green first
  - change mapped to explicit blocker class
  - then canary
  - then paid rerun only if the specific previously failing promptless-authoritative cases are now explicitly reclassified or resolved
- Keep one authoritative machine-readable gate-status source for this phase path; other summaries may explain it, but should not compete with it.

## Risks And Failure Modes

- Overfitting the comparator to the current 5-case canary instead of making it a small generic normalization layer
- Smuggling new benchmark-friendly phrasing back into the deterministic core
- Reclassifying evaluator ambiguity in a way that hides genuine kernel regressions
- Letting artifact sprawl create multiple contradictory “truth” files about the same gate state
