# Plan 03-03: Enforce Abstention, Token-Budget, And Delivery-View Rules

## Objective

Close the retrieval constitution by enforcing strict abstention triggers, answer-mode-specific budgets, high-risk temporal behavior, and delivery-view discipline.

## Must Haves

- token budget is a real answer contract, not only a `max_items` knob
- delivery views shape output without rewriting Core2 truth
