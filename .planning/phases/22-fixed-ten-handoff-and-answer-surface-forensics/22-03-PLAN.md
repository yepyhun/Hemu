# Plan 22-03: Conclude With One Canonical Downstream Action

## Goal

Reduce the fixed-ten downstream forensics to one canonical next action: one bounded fix direction or one explicit stop verdict.

## Tasks

- aggregate seam counts across the fixed ten
- identify whether one downstream seam family is dominant enough to justify a bounded implementation phase
- map each dominant seam family to one allowed next action
- reject ambiguous multi-direction carry-forward conclusions
- write the explicit stop verdict if no single justified downstream action exists

## Deliverables

- `22-DECISION-RUBRIC.md`
- `22-OUTCOME.json`
- `22-VERDICT.md`

## Verification

- the phase ends with exactly one canonical recommendation or stop verdict
- the recommendation is justified by seam counts and checkpoint evidence
- the conclusion does not silently widen into multiple speculative directions
