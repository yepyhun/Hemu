# Plan 21-02: Trace Memory Presence And Loss Seams

## Goal

For each fixed-ten case, determine the earliest lifecycle checkpoint at which answer-bearing memory is absent or lost.

## Tasks

- inspect memory/source presence for each case
- inspect ingestion/extraction traces where available
- inspect normalized/persisted representations where available
- inspect indexing / retrieval visibility
- inspect admission/prefilter loss
- inspect runtime evidence surface and handoff loss
- use old Hermes only as a diagnostic comparator when needed

## Deliverables

- `21-LIFECYCLE-LEDGER.jsonl`
- `21-SEAM-TAXONOMY.md`
- `21-OLD-HERMES-COMPARATOR.md`

## Verification

- comparator use stays diagnostic-only
- the ledger shows where the earliest loss happens, not just the final benchmark label
