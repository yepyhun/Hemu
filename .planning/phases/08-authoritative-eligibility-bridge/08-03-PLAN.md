# 08-03 Plan: Verdict And Next-Step Gate

## Goal

Translate the local bridge outcome into one bounded next-step decision without reopening loops.

## Tasks

1. Compare the local outcome against the frozen manifest and expected route-shift target.
2. Classify the result into one canonical verdict:
   - `go` — broader rerun is now justified
   - `hold` — keep hybrid as forward branch but do not rerun yet
   - `stop` — revert or contain the bridge and do not continue this direction
3. Record the next-step gate explicitly so later work cannot reinterpret the result loosely.

## Deliverables

- `08-VERDICT.md`
- `08-NEXT-STEP.md`
- updated `08-LOCAL-OUTCOME.json` if needed as the single truth source

## Verification

- Next-step language is derived from the canonical local outcome file, not a freeform narrative.
- No broader paid rerun is authorized unless the local bridge shows a real route-shift.
- If the result is `hold` or `stop`, the phase closes without spawning a micro-tuning loop.
